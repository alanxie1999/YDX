# 调整首次下注金额

## 方法一：使用命令（推荐）

在管理员控制台发送：

```bash
/ys 预设名 连续 止损 一输 二输 三输 四输 首注
```

### 示例

设置首注为 5000：
```bash
/ys 5k 1 10 3.0 2.5 2.2 2.1 5000
```

设置首注为 20000：
```bash
/ys 2w 1 10 3.0 2.5 2.2 2.1 20000
```

### 参数说明

| 参数 | 说明 | 示例 |
|------|------|------|
| 预设名 | 预设名称（自定义） | 5k, 1w, 2w |
| 连续 | 连续押注次数 | 1 |
| 止损 | 连输止损手数 | 10 |
| 一输 | 第 1 输倍率 | 3.0 |
| 二输 | 第 2 输倍率 | 2.5 |
| 三输 | 第 3 输倍率 | 2.2 |
| 四输 | 第 4+ 输倍率 | 2.1 |
| **首注** | **首次下注金额** | **5000** |

## 方法二：编辑预设文件

编辑 `users/你的账号名/presets.json`：

```json
{
    "5k": ["1", "12", "3.0", "2.5", "2.2", "2.1", "5000"],
    "1w": ["1", "11", "3.0", "2.5", "2.2", "2.1", "10000"],
    "2w": ["1", "10", "3.0", "2.5", "2.2", "2.1", "20000"]
}
```

**数组第 7 个元素（索引 6）是首注金额**。

## 方法三：启动预设

设置好后，使用 `/st` 命令启动预设：

```bash
/st 5k
```

## 查看当前预设

```bash
/yss
```

会显示所有可用预设及其参数，包括首注金额。

## 默认预设参考

| 预设名 | 首注金额 | 连输止损 |
|--------|---------|---------|
| 5k | 5,000 | 12 手 |
| 1w | 10,000 | 11 手 |
| 2w | 20,000 | 10 手 |
| 3w | 30,000 | 10 手 |
| 5w | 50,000 | 9 手 |
| 10w | 100,000 | 8 手 |

## 注意事项

1. **金额单位**：灵石（朱雀平台货币单位）
2. **最小金额**：建议至少 500 以上
3. **倍投链**：首注会影响后续倍投金额，确保资金充足
4. **修改后需要重启**：编辑 presets.json 后需要重启脚本才能生效
5. **命令修改立即生效**：使用 `/ys` 命令修改后立即生效，无需重启
